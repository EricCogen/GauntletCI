# Support

## Questions

For usage questions, open a GitHub issue using the most relevant template.

## Bugs

Use the [bug report template](https://github.com/EricCogen/GauntletCI/issues/new?template=bug_report.yml).

## False positives

Use the [false positive template](https://github.com/EricCogen/GauntletCI/issues/new?template=false_positive.yml).

## Risky diffs

Use the [risky diff template](https://github.com/EricCogen/GauntletCI/issues/new?template=risky_diff.yml).

## Security

Do not open public issues for vulnerabilities. Use [GitHub Security Advisories](https://github.com/EricCogen/GauntletCI/security/advisories/new).

## Before asking for help

Please include:

- GauntletCI version
- Operating system or CI environment
- Command used
- Relevant output
- Minimal sanitized diff if applicable
